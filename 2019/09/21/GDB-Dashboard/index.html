GDB Dashboard

Introduction

  • A .gdbinit script make GDB shows a visual dashboard of
    • source code
    • memory
    • register
    • userdefined watch expression
  • Get from here: https://github.com/cyrus-and/gdb-dashboard
  • Demostration:

Installation

  • put .gdbinit into home folder
  • GDB dashboard requires at least GDB 7.7 compiled with Python 2.7

Basic usage

  • inorder to use
    • gdb –args xrun *.v -tcl
  • command dashboard (or shorten as da) cause refresh of dashbaord
  • command help dashboard
