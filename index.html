Leetcode#403 Frog Jump

Posted on 2019-09-21 In leetcode Disqus:

思路： DFS + Prune

  1. 目標是跳到最後一個石頭 N
  2. 假設最後一步是從第 X 個石頭 (X= 1, 2, 3…N-1） 跳到第 N 個石頭，是否有可能？
    • 若 （第 N 個石頭與第 X 個石頭的距離差） < （第 X 個石頭的位置），就有可能
  3. 對於每個可能的石頭 X
    • 取其與第 N 個石頭的距離為 K (stones[N] - stones[X])
    • 再往前考慮 是否能從第 Y 個石頭，(Y= 1, 2, 3…X-1），以跳躍 （K-1, K, K+1) 其中一種距離，跳到第 X 個石頭？
      • 若是 stone[X] 與 stone[Y] 的距離是（K-1, K, K+1) 其中一個，則有可能，==進行遞迴==
        • 將 X 代換為 Y
        • 將 K 代換為 stone[X] - stone[Y]
        • 遞迴停止條件：找到第 0 個石頭, 並且 K 為 1
      • 若是 stone[X] 與 stone[Y] 的距離大於 K+1, 則不可能, 且比 Y 還要前面的石頭都不用考慮了(==剪枝==)
